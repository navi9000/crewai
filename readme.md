# О проекте

Набор тренировочных проектов по crewai.

## Настройка и запуск проекта

Перед запуском любого из проектов необходимо поместить в корень проекта файл .env со следующими параметрами: MODEL - имя используемой модели. Если используется платная модель, необходимо также указать ее ключ api. Например, для моделей от Open AI необходимо указать OPENAI_API_KEY.

В качестве альтернативы можно установить локальную бесплатную модель ollama. Для этого необходимо скачать продукт с сайта https://ollama.com, а затем запустить команду `ollama pull llama3.2`. В этом случае в качестве MODEL необходимо указать "ollama/llama3.2".

Для запуска какого-либо из проектов необходимо перейти в папку проекта и запустить команду запуска. Например, для проекта debate:
cd debate
crewai run
